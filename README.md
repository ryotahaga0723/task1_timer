# README

## 開発言語
- Ruby 3.0.1p64
- Rails 6.0.4.7

## 実行手順
> git clone https://github.com/ryotahaga0723/task1_timer.git  
> cd timer_app  
> bundle install  
> rails db:create && rails db:migrate  
> rails seed  
> rails s
