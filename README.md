# README

## 開発言語
- Ruby 3.0.1p64
- Rails 6.0.4.7

## 実行手順
> git clone https://github.com/ryotahaga0723/timer_app.git  
> cd timer_app  
> bundle install  
> yarn install --check-files
> rails db:create && rails db:migrate  
> rails db:seed  
> rails s
